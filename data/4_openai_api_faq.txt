Q: What is the OpenAI API?
A: The OpenAI API can be applied to virtually any task that involves understanding or generating natural language, code, or images. It offers a spectrum of models with different levels of power suitable for different tasks, including GPT-4, GPT-3.5, and DALL-E.

Q: How is the API priced?
A: Pricing is based on a pay-as-you-go model. For text models, you are billed per 1,000 tokens (which is roughly 750 words). Prices vary depending on the specific model used (e.g., GPT-4 is more expensive than GPT-3.5-turbo). Image models are billed per image generated.

Q: What are tokens?
A: Tokens are pieces of words used for natural language processing. For typical English text, one token is about 4 characters or 0.75 words. You can use the OpenAI Tokenizer tool to see how a text string is broken down into tokens.

Q: Is there a free tier for the API?
A: New users receive a small amount of free credit to experiment with the API. Once this credit expires or is exhausted, you must enter billing information to continue using the API.

Q: What is the rate limit for the API?
A: Rate limits define how many requests or tokens you can send per minute (RPM and TPM) and per day (RPD). Rate limits depend on your usage tier, which is determined by the total amount you have paid and the time passed since your first successful payment.

Q: How do I manage my API keys?
A: You can generate and manage API keys from the API keys page in your OpenAI dashboard. It is highly recommended to keep your keys secure and not share them in public repositories or client-side code. Use environment variables or a secrets manager to store them safely.

Q: Can I use the API for commercial purposes?
A: Yes, you can use the OpenAI API in commercial products. You own the input and output generated through the API, subject to OpenAI's Terms of Use and usage policies.

Q: Are my API inputs used to train OpenAI's models?
A: No, by default OpenAI does not use data submitted to and generated by the API to train their models. Your data remains private to your organization.
